# PhobosLT 4ch — ESP32 Embedded Project

This is an **ESP32 embedded firmware project**. There is no local web dev server and no browser-based preview workflow.

- Do NOT use `preview_start`, `preview_screenshot`, or any `preview_*` tools.
- Do NOT attempt to start a dev server.
- Verification is done by flashing firmware to the device via PlatformIO (`pio run --target upload`).
- After editing firmware files, no preview verification step is needed or possible.
